# Secure POS Pro Ultimate UI Version

## English
A Bootstrap-based secure POS system for mini markets and shops using PHP and MySQL.  
Includes dashboard, product management, customer management, supplier management, stock in/out, barcode placeholder, PDF receipt placeholder, and Burmese/English ready UI.

## မြန်မာ
ဒီ project သည် PHP + MySQL + Bootstrap အသုံးပြုထားသော Secure POS Pro Ultimate UI Version ဖြစ်ပါသည်။  
Mini market နှင့် shop များအတွက် dashboard, product management, customer management, supplier management, stock in/out, barcode placeholder, PDF receipt placeholder တို့ပါဝင်ပါသည်။ 

## Features
- Secure login
- Bootstrap admin UI
- Products management
- Customers management
- Suppliers management
- Stock in / Stock out
- Sales checkout structure
- Barcode scanner placeholder
- PDF receipt placeholder
- Burmese / English ready UI
- MySQL database structure

## Setup
1. Install XAMPP / Laragon / WAMP
2. Put folder into `htdocs`
3. Import `database.sql`
4. Open `http://localhost/secure-pos-pro-ultimate-ui/auth/login.php`
